## ğŸ Getting Started

### Prerequisites
*   Node.js (v18+)
*   Expo Go app on your physical device (Android/iOS)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/JDanielZ5G/Evoting_mobile_group3.git
    cd Evoting_mobile_group3
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the app:**
    ```bash
    npx expo start
    ```

4.  **Run on Device:**
    *   Scan the QR code with the **Expo Go** app (Android) or Camera app (iOS).
